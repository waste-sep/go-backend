## Rules
Branch's name format: `<type>/<name>`; e.g. `feature/core-components`
- Types:
  - feature: new feature introduced
  - fix: bug fixes
  - fefactor: refactor existing codes
  - docs: add docs
  - chore: chore
- When committing, please use https://gitmoji.dev/

## Execution
- To run the code, type `go run .`
